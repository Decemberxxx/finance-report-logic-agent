# finance-report-logic-agent
我构建了一个面向投研场景的「金融研报→一页投资逻辑图」自动化 Agent，核心解决券商 PDF 研报信息冗余、人工提炼效率低下、投研逻辑难以标准化沉淀的痛点。此前投研人员处理一份 30-50 页的个股 / 行业研报，需耗时 30 分钟以上精读、标记关键数据、手动整理思维导图，过程中易遗漏核心逻辑，且输出格式不统一，不利于复盘与分享。
该 Agent 的核心逻辑流为完整的自动化闭环：首先通过 PDF 解析与文本清洗模块提取研报全文并过滤页眉页脚、图表乱码等无效信息；再通过大模型进行长链结构化推理，按投研标准 7 大维度（标的、赛道、核心逻辑、利好、风险、估值、结论）进行多轮提炼与校验，确保信息不遗漏、不冗余；最后自动将结构化信息转换为标准 Mermaid 思维导图代码，并生成可直接复用的 Markdown 文档。
目前该 Agent 已在个人投研复盘场景落地，累计处理 50 + 份研报，单份处理时间从人工 30 分钟压缩至 1 分钟以内，单份研报处理平均消耗约 15 万 Token，信息提炼准确率达 90% 以上，大幅提升了投研信息处理效率；输出的标准化逻辑图可直接用于复盘笔记、投资周报等场景，显著降低了投研工作的重复性成本。
